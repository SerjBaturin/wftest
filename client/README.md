### Commit structure
path required folder/file.js - *
  - ADD: add file, folder, module or function *
  - FIX: fix issue *
  - DEL: remove folder or file *
  - REF: refactoring (items required)
  - ENV: add plugins, libs & etc 
  - UNDEF: undefined changes *
